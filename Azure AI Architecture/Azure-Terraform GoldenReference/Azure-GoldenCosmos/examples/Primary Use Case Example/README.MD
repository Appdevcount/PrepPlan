# Cosmos DB Primary Use Case example

> This example shows how to deploy the following:
>
> * Cosmos Account
> * Cosmos Database
> * 4 Cosmos containers where each container using specific Index, Throughput and Scaling settings
> * Firewall Settings (allowed subnets, IPs and portal access)
> * Automatic Failover
> * Failover Geo Locations with Priority
> * Required Tags
> * Several default alerts integrated with alarm funnel
> * Diagnostic Settings
> * Private Endpoint
