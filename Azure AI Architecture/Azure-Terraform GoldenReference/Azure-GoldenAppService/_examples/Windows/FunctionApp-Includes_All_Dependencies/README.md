# Function App - Windows example

> This example shows how to create one Windows Function App.
> Includes all Function App dependency resources.
>
> * Windows Function App
>   * Application Stack: .Net v8.0
>   * VNet Integration
>   * Private Endpoint
>   * Custom App Settings
>   * Sticky Settings (for use with Slots)
>   * CORs support
>   * IP restrictions including SCM IP restrictions
>   * App Scaling
> * App Service Plan
>   * Using Elastic Premium Plan for Dynamic Scaling
> * Storage Account for Function App
> * Function App Slot
> * Managed Identity Role assignment between Function App and Storage Account
> * App Insights
> * * Includes Log Analytics Workspace
> * Alerts
>   * Includes Alarm Funnel Action Group
>   * User defined Action Groups
> * Required Tags
> * Diagnostic Settings
> * Private Endpoint
>   * For Function App
>   * For Storage Account
