# VNET Flow Log  and Diagnostics using Storage Account with Private Endpoint and CMK example

> This example shows how to deploy the following:
>
> * Vnet\Subnets
> * Storage Account for Flow Log and\or VNet Diagnostics
> * Storage Account CMK
> * Role Assignment for CMK
> * Flow Log
> * NSG
> * Diagnostic Settings
> * Private Endpoint for Storage Account
> * Required Tags
>
> NOTE: When using this example, be sure to submit a request for VNet Peering. The Private Endpoint created in this example will not take effect until VNet peering with CCOE managed hub is completed.
