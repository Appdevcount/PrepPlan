
# How to Use This Guide with GHCP and Vibe Coding

This ZIP package contains the cleaned .NET-only REST API development guide for .NET Core 8.0.

## Files Included
- REST_API_DotNetCore_8.0_Cleaned.md: The main guide with only .NET-specific content.

## Usage Instructions

### GitHub Copilot Chat (GHCP)
1. Add the markdown file to your repository (e.g., `/docs/REST_API_Guide.md`).
2. Reference it in your prompts:
   > "Follow the standards from `REST_API_Guide.md` while generating this controller."
3. Optionally, configure `.copilot/config.json` to include this file as context.

### Vibe Coding
1. Upload or pin this file as a reference document.
2. Use it to validate API design, testing, and architecture decisions.
3. Ensure all coding follows the mandatory thinking process and checklist.

This guide ensures consistency, performance, and compliance across all .NET REST API implementations.
